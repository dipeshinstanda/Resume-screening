## 🆘 **HELP & SUPPORT**

### **Quick Fixes**

**"Python not found"**
→ Install from https://www.python.org/downloads/

**"Node not found"**
→ Install from https://nodejs.org/

**"ImportError: cannot import name 'etree' from 'lxml'"** ⭐ COMMON
→ Run: `.\FIX_LXML_ERROR.ps1` (see `LXML_FIX_COMPLETE.md`)

**"Port already in use"**
→ Run: `taskkill /F /IM python.exe; taskkill /F /IM node.exe`

**"Dependencies missing"**
→ Run: `cd backend; py -m pip install -r requirements.txt; cd ..\frontend; npm install`
