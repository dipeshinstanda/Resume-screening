# Research: AI-Based Resume Screening System

## Project Information

**Organization**: EmpowerTech Solutions  
**Location**: Chennai, Tamil Nadu, India  
**Project Type**: Research and Development

## Objectives

1. To design and develop an AI system capable of screening resumes based on the education background of candidates
2. To integrate machine learning algorithms for accurate matching of educational qualifications with job requirements
3. To assess the effectiveness of the system in improving recruitment processes for educational institutions

## Research Tasks

### 1. Literature Review
- Research existing AI technologies used in resume screening
- Study current machine learning approaches for document analysis
- Review NLP techniques for education qualification extraction

### 2. Algorithm Development
- Develop algorithms for analyzing educational qualifications
- Implement matching algorithms based on job requirements
- Create scoring and ranking mechanisms

### 3. Data Collection
- Collect sample resumes (with proper consent)
- Gather job descriptions from educational institutions
- Create training and testing datasets

### 4. Model Development
- Text extraction from PDF and DOCX files
- NLP-based education information extraction
- TF-IDF vectorization for text similarity
- Cosine similarity for matching
- Degree hierarchy classification

### 5. Evaluation Metrics
- **Accuracy**: Measure overall correctness
- **Precision**: Ratio of true positives to predicted positives
- **Recall**: Ratio of true positives to actual positives
- **F1 Score**: Harmonic mean of precision and recall

### 6. Documentation
- System architecture documentation
- API documentation
- User guide
- Research paper for publication

## Technical Approach

### Education Extraction
- Pattern matching for degree identification
- Field of study extraction
- Institution name recognition
- Year extraction

### Matching Algorithm
1. Text preprocessing
2. TF-IDF vectorization
3. Cosine similarity calculation
4. Degree hierarchy matching
5. Final score calculation (60% similarity + 40% degree match)

### Degree Hierarchy
- PhD/Doctorate: Level 5
- Masters/MBA: Level 4
- Bachelors: Level 3
- Diploma/Associate: Level 2
- High School: Level 1

## Expected Outcomes

1. Functional AI system for resume screening
2. Improved efficiency in candidate evaluation
3. Reduced bias in initial screening
4. Quantified performance metrics
5. Published research paper
