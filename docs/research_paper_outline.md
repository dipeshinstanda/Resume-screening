# Research Paper: AI-Based Resume Screening System for Educational Qualifications

## Abstract
This paper presents the development and evaluation of an AI system designed to screen resumes based on candidates' educational backgrounds. The system utilizes natural language processing and machine learning techniques to match educational qualifications with job requirements, aiming to improve recruitment processes in educational institutions.
